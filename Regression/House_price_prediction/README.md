# House Price Prediction

### Project Overview

This project focuses on predicting house prices using  machine learning regression model.
The dataset contains information such as number of bedrooms, bathrooms, square footage, location details, and construction year.

#### The goal is to:

Perform data preprocessing & EDA

Analyze correlation with the target variable

Train and evaluate  regression models

Improve performance using transformations

### Data Preprocessing Steps

Handling missing values

Selecting numeric-only features

Log transformation for skewed data

Train-test split

Correlation analysis with target column
